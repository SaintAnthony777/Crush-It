# Crush It by RADAHY Anthony 
Une application assez simple codé en godot engine, il permet de compresser les fichiers textes<br>
***À noter que l'exécutable ne fonctionne que sous linux***
# Comment utiliser ?
Très simple, il suffit de lancer l'executable, "Crush it beta version.x86_64", suite à cela une interface s'ouvrira, et il suffit de cliquer sur le boutton ouvrir un fichier, suite à cela une boîte de dialogue s'ouvrira et il faut selectionner le fichier texte, après, le bouton compresser sera valide si la taille du fichier est acceptable (dépasse les 100MO).<br>
# Sorties :
Une fois la compression lancée, il n'y a pas encore d'affichage de progression en cours mais une fois la compression terminée, il y aura un texte qui s'affichera en bas du boutton compresser.<br>
Le fichier de sortie sera dans le même repertoire que le fichier source, à l'exception du fait que le fichier aura _COMPRESSED.bin à la fin et le fichier decompréssé aura _DECOMPRESSED.txt à la fin.
